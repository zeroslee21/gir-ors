# ORS 탐색기 - 제품 문서

## 개요

한국 온실가스종합정보센터(GIR) 상쇄등록부시스템(ORS)에 등록된 314건의 감축방법론을 탐색하고 분석할 수 있는 인터랙티브 웹 플랫폼.

## 핵심 가치

- 314건 감축방법론의 직관적 탐색 및 검색
- CDM/국내 방법론 비교 분석
- 15개 사업분야별 체계적 분류 시각화
- 제도 변화(제4차 기본계획, CDM 종료, 파리협정 제6조)의 맥락적 이해

## 타겟 사용자

- 배출권거래제 참여 기업 담당자
- 외부사업 검증기관
- 정책 연구자 및 컨설턴트
- 탄소시장 관련 학술 연구자

## 주요 기능

### 방법론 탐색 (/methodologies)
- 314건 전체 데이터 테이블
- 퍼지 검색 (Fuse.js)
- 다중 필터링: 유형, CDM분류, 사업분야, 관장기관, 상태
- 페이지네이션

### 방법론 상세 (/methodologies/[id])
- 4대 구성요소: 적용조건, 베이스라인, 모니터링, 감축량 산정
- ER = BE - PE - LE 수식 시각화
- 관련 방법론 추천

### 통계 대시보드 (/statistics)
- CDM vs 국내 분포 파이 차트
- CDM 분류별(AM/ACM/AMS) 파이 차트
- 사업분야별 방법론 수 막대 차트
- 연도별 증가 추이(2015-2024) 라인 차트

### 사업분야 (/sectors)
- 15개 UNFCCC Sectoral Scope 분류
- 산업부문별 vs 배출원기반 구분
- 분야별 방법론 목록

### 관장기관 (/agencies)
- 5개 부처: 환경부, 산업통상자원부, 국토교통부, 농림축산식품부, 해양수산부
- 기관별 관할 방법론 현황

### 제도 안내 (/guide)
- 배출권거래제 외부사업 개요
- 방법론 4대 구성요소 해설
- 추가성 입증 절차
- CDM 체계 및 제도 전망

## 데이터 출처

- ORS 상쇄등록부시스템 (ors.gir.go.kr)
- 한국에너지공단 외부사업 포털 (offset.energy.or.kr)
- UNFCCC CDM 방법론 목록 (cdm.unfccc.int)
- 환경부 고시 제2023-258호

## 배포

- **URL**: https://ors.verdex.kr
- **인프라**: AWS EC2 (t3.micro) + ALB + Nginx
- **SSL**: ACM (ALB 종료)
