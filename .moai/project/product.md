# GIR ORS 제품 문서

## 프로젝트 개요

### 이름 및 설명

**GIR ORS (Global Initiatives Registry - Offset Registry System) 방법론 탐색기**

한국의 온실가스 감축 방법론을 체계적으로 탐색하고 비교할 수 있는 웹 애플리케이션입니다. 국제 탄소 감축 표준(CDM: Clean Development Mechanism)과 국내 정책 기반 감축 방법론 314건을 데이터화하여 관리 및 검색 기능을 제공합니다.

### 프로젝트 배경

한국의 감축 방법론은 국제 기준(UNFCCC CDM)과 국내 정책(MOE, MOTIE, MOLIT, MAFRA, MOF)에서 제시되고 있으나, 통합 관리 체계가 부족했습니다. 본 프로젝트는 이러한 분산된 방법론 정보를 통합하여:

- 정책 입안자: 기존 정책의 방법론 커버리지 확인 및 정책 간 중복 분석
- 기업 ESG 담당자: 자사에 적용 가능한 감축 방법론 발굴
- 연구 기관: 감축 방법론의 분포, 추세, 효과성 분석
- 개발자: 탄소 감축 기술의 시장 현황 이해

이를 통해 한국의 감축 노력이 국제 기준과 어떻게 정렬되어 있는지 가시화합니다.

## 핵심 가치 제안

### 통합 관리

314건의 감축 방법론을 단일 플랫폼에서 관리합니다.
- CDM 기반 211건: 국제 표준 준수
- 국내 정책 기반 103건: 한국 고유 정책 반영

### 고급 검색 및 필터링

Fuse.js 기반 퍼지 검색으로 방법론명, 설명, 번호를 한 번에 검색합니다.
다차원 필터링:
- 부문(15개 UNFCCC 섹터)
- 출처(CDM vs 국내 정책)
- 주관 기관(MOE, MOTIE, MOLIT, MAFRA, MOF)

### 시각적 분석

- 섹터별 분포 그리드: 15개 부문의 방법론 수를 시각화
- 통계 대시보드: 4종 Recharts 차트로 시계열, 분포, 비교 제공
- 정책별 구성: 5개 주관 기관의 포트폴리오 표시

### 상세 문서화

각 방법론마다:
- 공식 정의 및 적용 범위
- 계산 공식 (LaTeX 렌더링)
- 요구사항 및 제약사항
- 참고 자료 및 링크

### 북마크 기능

사용자별 맞춤형 관심 방법론 저장 및 관리.

## 타겟 사용자 (페르소나 4종)

### 1. 정책 입안자 (Policy Maker)

**역할:** 중앙 부처 및 지자체의 기후정책 담당자

**목표:**
- 기존 정책의 방법론 커버리지 확인
- 정책 간 중복 또는 갭 분석
- 국제 기준과의 정렬 여부 평가
- 신규 정책 수립 시 참고 자료

**사용 패턴:**
- 월 1-2회 방문
- 부문별 필터링으로 정책 매핑
- 통계 대시보드로 전체 그림 파악
- 상세 방법론 문서 다운로드

**가치:** 정책 중복 제거, 효율성 증대, 국제 기준 준수

### 2. 기업 ESG 담당자 (Corporate ESG Manager)

**역할:** 대규모 기업의 지속가능성 및 ESG 전략 담당자

**목표:**
- 기업 활동에 적용 가능한 감축 방법론 발굴
- 감축량 정량화 및 검증 가능성 확인
- 정부 인센티브 수령 가능성 판단
- 적용 기업 사례 및 유사 기업 분석

**사용 패턴:**
- 주 1-2회 방문
- 부문/기관 필터로 적용 가능 방법론 탐색
- 북마크로 후보 방법론 저장
- 상세 문서로 요구사항 분석

**가치:** 감축 계획 수립 가속화, 규제 준수, 인센티브 활용

### 3. 연구 기관 분석가 (Research Analyst)

**역할:** 대학, 출연연구소의 탄소 중립 및 감축 연구자

**목표:**
- 감축 방법론의 분포, 추이, 효과성 분석
- 기술별 방법론 현황 파악
- 정책 효과 평가 및 개선 제안
- 국제 비교 및 벤치마킹

**사용 패턴:**
- 주 1-3회 장시간 사용
- 통계 대시보드로 트렌드 분석
- 다중 필터로 부분집합 추출
- 데이터 다운로드 및 학술 활용

**가치:** 연구 기초 데이터 제공, 분석 시간 단축, 학술 발간

### 4. 기술 개발자 (Tech Developer)

**역할:** 탄소 감축 기술 스타트업, 솔루션 벤더의 개발자

**목표:**
- 탄소 감축 시장의 현황 이해
- 기술 적용 가능 부문 파악
- 규제 및 표준 요구사항 숙지
- 시장 진출 전략 수립

**사용 패턴:**
- 월 1회 정도 방문
- 특정 부문/기관별 상세 검색
- 상세 방법론 문서 참고
- API 또는 데이터 다운로드 활용

**가치:** 시장 기회 분석, 제품 개발 방향 설정, 규제 대응

## 주요 기능 (8개 페이지별 상세)

### 1. 홈 페이지 (/)

**구성:**
- 헤더: 프로젝트 로고, 타이틀, 주요 통계
- 통계 카드: 총 방법론 수(314), CDM 수(211), 국내 정책(103), 주관 기관(5)
- 섹터 그리드: 15개 UNFCCC 섹터별 방법론 수 시각화
- 정책 요약: 5개 정부 부처 소개 및 링크
- 공식 렌더링: 주요 감축 계산 공식 LaTeX 표시
- 네비게이션: 주요 메뉴로 이동

**역할:** 프로젝트 소개 및 사용자 가이드

### 2. 방법론 목록 (/methodologies)

**기능:**
- **검색:** Fuse.js 퍼지 검색 (방법론명, 설명, 번호)
- **필터링:** 부문, 출처, 주관 기관
- **페이지네이션:** 20개 항목 단위 표시, 클라이언트 기반 처리
- **정렬:** 방법론 ID, 이름, 부문별 정렬

**테이블 컬럼:**
- ID: 고유 식별자
- 이름: 방법론 공식명
- 부문: UNFCCC 섹터 분류
- 출처: CDM 또는 정책 주관 기관
- 상태: 활성 여부

**기술 구현:**
- Server Component: 초기 데이터 로딩
- Client Component: 검색, 필터, 페이지네이션 로직
- Fuse.js: 퍼지 검색 엔진 (threshold: 0.3)

### 3. 방법론 상세 (/methodologies/[id])

**기능:**
- 방법론 공식명, ID, 부문, 주관 기관 표시
- 방법론 설명 (개요, 적용 범위, 제약사항)
- 감축량 계산 공식 (LaTeX 렌더링)
- 요구사항 및 필수 데이터
- 참고 자료 및 외부 링크
- 북마크 토글 버튼
- 관련 방법론 추천 (같은 부문)
- 이전/다음 방법론 네비게이션

**기술 구현:**
- SSG 생성: 256개 경로 (각 방법론)
- 동적 라우트: getStaticPaths, getStaticProps
- 마크다운 → HTML 변환
- LaTeX 렌더링 (react-katex)

### 4. 부문 목록 (/sectors)

**기능:**
- 15개 UNFCCC 섹터 그리드 표시
- 각 섹터별 방법론 수 표시
- 섹터 아이콘 및 설명
- 섹터 클릭 → 상세 페이지로 이동

**시각화:**
- CSS Grid 반응형 레이아웃
- 카드: 섹터명, 방법론 수, 아이콘
- 호버 효과: 상세 설명 팝업

### 5. 부문 상세 (/sectors/[id])

**기능:**
- 부문 공식명, 설명, 관련 활동 설명
- 해당 부문의 방법론 테이블
- 필터링 (출처, 기관)
- 부문별 통계

**기술 구현:**
- SSG 생성: 15개 경로 (각 부문)
- 부문별 방법론 필터링

### 6. 통계 대시보드 (/statistics)

**기능:**
- 4종 Recharts 차트:
  1. **파이 차트:** CDM vs 국내 정책 비율
  2. **막대 차트:** 기관별 방법론 수
  3. **선 그래프:** 연도별 추가된 방법론 수 (2015-2024)
  4. **레이더 차트:** 부문별 방법론 분포

**인터랙션:**
- 범례 클릭으로 데이터 시리즈 토글
- 호버 시 값 표시
- 반응형 크기 조정

**기술 구현:**
- Client Component (Recharts)
- 정적 데이터 (JSON)

### 7. 주관 기관 (/agencies)

**기능:**
- 5개 정부 부처 소개
  - MOE (환경부): 39개 방법론
  - MOTIE (산업부): 30개 방법론
  - MOLIT (국토부): 20개 방법론
  - MAFRA (농림부): 12개 방법론
  - MOF (재정부): 6개 방법론
- 각 기관별 설명, 담당 영역, 관련 정책
- 기관별 방법론 목록 링크

### 8. 정책 교육 가이드 (/guide)

**기능:**
- 온실가스 감축의 기초 개념 설명
- CDM 제도의 역사 및 현황
- 한국의 감축 정책 진화 (2015-2024)
- 방법론 활용 사례
- FAQ: 자주 묻는 질문
- 리소스: 관련 링크, 문서, 용어집

- CDM 방법론: UNFCCC 공식 데이터 (2024년 10월 기준)
